# Network Settings

## Purpose
Documents IP configuration, DNS, and firewall settings for the PBX system.

## Example
- IP Address: 192.168.1.10
- Subnet Mask: 255.255.255.0
- Gateway: 192.168.1.1
- DNS: 8.8.8.8, 8.8.4.4

## Notes
Ensure the PBX has a static IP and proper port forwarding for SIP and RTP.
